「参考」「見送り」のリストレイアウトを修正
script.js では、createArticleCard でカード型のHTMLを組み立てていますが、rec-content が compact-list の場合は別のテンプレートが使用されているため、点線付きの横長リストになっています。compact-list 用のHTMLを次のように変更してください。

// rec-contentがcompact-listの場合の各記事テンプレート
function createCompactListItem(article) {
  const li = document.createElement('div');
  li.className = 'compact-item';
  const rec = getLabelFor(article);
  li.innerHTML = `
    <div class="compact-card ${rec}">
      <span class="label-pill rec-${rec}">${getLabelText(rec)}</span>
      <a href="${escapeHtml(article.url)}" class="compact-title">${escapeHtml(article.title)}</a>
      <div class="compact-meta">
        <span class="compact-source">出典: ${escapeHtml(article.sourceDomain || extractDomain(article.url) || article.source)}</span>
        <span class="compact-date">・${formatRelativeDate(article.publishedAt || article.published_date)}</span>
      </div>
    </div>
  `;
  return li;
}


これに対応するCSSを styles.css に追加し、点線や縦書きラベルを完全に削除し、背景色やボーダーを整えたリストカードとして表示します。

.compact-item { margin-bottom: 0.75rem; }

これにより、「参考」「見送り」のリストはカード型の要素として表示され、ラベルは左上に配置されます。

既存の点線と縦書きラベルを削除
現在の点線スタイルや縦書きラベルを設定しているCSS（おそらく .compact-list li など）は不要になるため削除するか無効化します。

ラベルピルのスタイル調整
.label-pill.rec-consider と .label-pill.rec-skip のスタイルが適用されていない場合は、styles.css 内で定義しているグラデーション背景と文字色がちゃんと当たるようクラス名の付与を確認してください
raw.githubusercontent.com
。
